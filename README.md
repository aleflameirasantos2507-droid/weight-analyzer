# Weight Analyzer

A beginner-friendly Python project that collects five weight values and identifies the highest and lowest weights entered.

## Features

- Accepts five weight values as input
- Tracks the highest weight entered
- Tracks the lowest weight entered
- Displays the final results
- Uses loops and conditional statements

## Technologies Used

- Python 3

## How to Run

1. Clone this repository:

```bash
git clone https://github.com/your-username/weight-analyzer.git
```

2. Navigate to the project folder:

```bash
cd weight-analyzer
```

3. Run the script:

```bash
python main.py
```

## Example

### Input

```text
What is the 1st weight? 72
What is the 2nd weight? 65
What is the 3rd weight? 81
What is the 4th weight? 70
What is the 5th weight? 68
```

### Output

```text
The highest weight is 81.0 and the lowest weight is 65.0
```

This project demonstrates:

- `for` loops
